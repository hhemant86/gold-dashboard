# Gold Price Analysis - Version 1

This project analyzes historical XAU/USD (gold) prices using Python,
Pandas, and Matplotlib.

### Features (v1)
- Loads daily gold prices from a CSV file
- Plots gold price trend
- Shows last 5 entries in terminal

### Tech Used
Python, Pandas, Matplotlib

### Upcoming Versions:
- Moving Averages
- Trend Analysis
- Buy/Sell Signal Generation
- Machine Learning Prediction
